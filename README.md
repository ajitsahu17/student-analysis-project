# 📊 Student Marks Analysis using Core Python

A simple Python project that analyzes student marks from a CSV file using core Python concepts such as file handling, lists, dictionaries, functions, and conditional statements.

---

## 🚀 Features

* Read student data from a CSV file
* Calculate total marks and average
* Classify students as Pass/Fail
* Display student performance
* Practice real-world Python data handling

---

## 🛠️ Technologies Used

* Python
* CSV File Handling
* Core Python Concepts

---
